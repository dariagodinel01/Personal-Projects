# NewLanguages: Chatbot-Assisted Language Learning

**NewLanguages** is a web application that makes learning a new language more interactive. It uses a real-time chatbot and speech recognition to help users practice speaking and get immediate feedback.

---

## Key Features

* **Interactive Chatbot:** Practice conversation with an AI bot that provides instant support.
* **Personalized Dashboard:** Displays user name and recent quiz scores upon login to track progress.
* **Level-Based Recommendations:** Suggests specific learning materials based on your previous quiz results.
* **DELF Exam Support:** Includes a speech recorder for students to practice and review their speaking skills.
* **Social & Navigation Tools:** Easy access to study modules, a community forum and a "Top" button for simple navigation.

---

##  Tech Stack

* **Backend:** Java / Spring Boot
* **Database:** MySQL (Stores user data, scores, and progress securely)
* **Security:** Encrypted data and secure user authentication.
* **Frontend:** HTML, CSS and JavaScript.
* **Chabot:** Frontend (HTML/JavaScript/CSS) and Backend (Flask/Python).

---

## Future Goals

* **More Languages:** Expanding support to Japanese, Spanish and Arabic.
* **Gamification:** Adding game-based quizzes to keep users motivated.
* **VR Practice:** Practicing conversations in Virtual Reality with human-like bots.
